# TACE AI Instructions

This repository is governed by the following authoritative documents, in descending order of authority:

1. docs/constitution/TACE_PHILOSOPHICAL_CONSTITUTION.md
2. docs/constitution/TACE_SEMANTIC_CONSTITUTION.md
3. docs/constitution/TACE_SOFTWARE_CONSTITUTION.md

These documents override the model's prior knowledge.

Before answering any question about TACE:

1. Search the constitutions.
2. Search the project source code.
3. If the answer exists, answer only from those sources.
4. If the answer does not exist, answer exactly:
   "Not defined in the current TACE project."
5. Do not infer missing ontology.
6. Do not redefine TACE terminology.
7. Clearly distinguish:
   - project facts,
   - implementation suggestions,
   - external knowledge.

The objective is to preserve the integrity of the TACE project rather than to maximize answer completeness.
